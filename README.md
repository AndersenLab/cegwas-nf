# cegwas-nf

Perform mapping of input.tsv


### Usage

```
# cd to directory containing a trait file.
nextflow run Andersenlab/cegwas-nf --in=<input file>
```

### Input Format

Save trait data as a `.tsv`; Strains in column 1. Traits are in columns 2 and above.
